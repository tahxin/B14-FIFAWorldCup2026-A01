# FIFA World Cup 2026 Hub

A clean, responsive, and fully semantic static landing page designed for the historic 2026 FIFA World Cup featuring 48 nations. Built entirely from scratch using pure HTML5 and CSS3 to replicate a modern sports entertainment UI layout.

## Features

*   **Hero Section:** High-impact visual header showcasing the tournament's core slogan: *"One Game. 48 Nations. One World."* with a call-to-action button.
*   **Key Tournament Metrics:** A statistics bar displaying ticket milestones, capacity percentages, and fan satisfaction data using modern CSS layout techniques.
*   **Host City Explorer:** A structured layout profiling key host cities (New York/New Jersey, Mexico City, Toronto, and Vancouver) complete with custom photography and essential metadata lists.
*   **Match Schedule Tracker:** A clean, accessible, and responsive data table mapping out highly anticipated fixtures, dates, venues, and groups.

## Tech Stack

*   **HTML5:** Semantic structural markup.
*   **CSS3:** Custom styling featuring responsive typography, modern alignment techniques, and layout rules for multi-device compatibility.

## Getting Started

Because this project relies strictly on native web technologies, it requires no compilation, build tools, or complex installations.

### Prerequisites
You only need a modern web browser (Chrome, Firefox, Safari, Edge) and a basic text editor.

### Installation & Launch

1. Clone or download the repository:
```bash
git clone https://github.com/tahxin/B14-FIFAWorldCup2026-A01.git

```


2. Navigate into the project folder:
```bash
cd B14-FIFAWorldCup2026-A01

```


3. Launch the project:
* **Option A:** Simply double-click the `index.html` file to open it in your default browser.
* **Option B:** If using VS Code, right-click `index.html` and select **Open with Live Server** for a live-reloading preview.
